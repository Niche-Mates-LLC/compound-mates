# Compound Mates

Shared AI toolset for Niche Mates engineering team.

## Installation

**Inside Claude Code (slash commands):**
```
/plugin marketplace add Niche-Mates-LLC/compound-mates
/plugin install compounds@compound-mates
```

**Or via CLI:**
```bash
claude plugin marketplace add Niche-Mates-LLC/compound-mates
claude plugin install compounds@compound-mates
```

## Commands

| Command | Description |
|---------|-------------|
| `/pesquisas` | Deep exploration and research before acting |
| `/review` | Review a PR focusing on real issues |
| `/deslop` | Remove AI slop and LLM generation fingerprints from branch changes |
| `/commit-or-push` | Commit (and optionally push) changes with proper message style |

## Contributing

Add new commands to `plugins/compounds/commands/`.
